# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1. In short: be respectful, be constructive, and assume good faith.
Harassment or exclusionary behavior of any kind is not tolerated.

Report unacceptable behavior to `conduct@privyq.dev`. Maintainers are obligated
to respect the privacy and security of reporters.
